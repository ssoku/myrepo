# じゃんけん対戦 (Rock Paper Scissors Battle)

A beautiful and interactive Rock-Paper-Scissors game built with React.

## Features

- 🎮 Interactive gameplay with computer opponent
- 📊 Score tracking (wins, losses, draws)
- 📜 Game history display
- 🎨 Beautiful modern UI with animations
- 📱 Responsive design for mobile and desktop
- 🌏 Japanese language interface

## How to Play

1. Choose your move: グー (Rock), パー (Paper), or チョキ (Scissors)
2. The computer will make its choice
3. See the result and updated scores
4. View your recent game history
5. Reset the game anytime to start fresh

## Game Rules

- グー (Rock) beats チョキ (Scissors)
- パー (Paper) beats グー (Rock)  
- チョキ (Scissors) beats パー (Paper)
- Same choices result in a draw (あいこ)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Technologies Used

- React (Hooks)
- CSS3 (Flexbox, Grid, Animations)
- Modern JavaScript (ES6+)

Enjoy playing じゃんけん対戦! 🎉
